# Newsfeed

This small React project demonstrates a simple social feed similar to a Twitter timeline. Posts are loaded from [JSONPlaceholder](https://jsonplaceholder.typicode.com) and displayed in a clean timeline UI.

## Features

- Shows posts from the JSONPlaceholder API
- Randomizes the order of posts on load
- Pagination to browse the timeline
- Search box for filtering posts by text
- Navigate to an individual post page to read details and comments

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
